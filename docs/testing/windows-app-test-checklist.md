# Windows デスクトップアプリ 手動テストチェックリスト

> 最終更新: 2025-01-XX  
> 目的: Windows C++ デスクトップアプリケーションの全機能動作確認チェックリスト

---

## 使い方

このチェックリストは、Windows デスクトップアプリ（C++）の全機能が正常に動作することを確認するためのものです。  
各テストケースを実施し、結果欄の ☐ を ☑（PASS）または ☒（FAIL）に変更してください。

各テストケースには以下の項目が含まれます：
- **テストID**: テストケースの一意識別子
- **テスト名**: テストの概要
- **前提条件**: テスト実施前に満たすべき条件
- **手順**: テスト実施手順（番号付き）
- **期待結果**: テスト成功時に観測されるべき結果
- **備考**: 補足情報・注意事項

## 1. 環境準備

### 1.1 テスト環境要件

| 項目 | 要件 |
|------|------|
| OS | Windows 10 以上 |
| Backend | Mobile BFF が起動していること（Port 8081） |
| データベース | PostgreSQL が起動し、初期データが投入されていること |
| テストユーザー | user001 / password123 |
| ネットワーク | localhost でBFFにアクセス可能であること |

### 1.2 事前確認チェックリスト

- [ ] Windows 10 以上の環境が準備されている
- [ ] PostgreSQL が起動している（`docker ps` で確認）
- [ ] Web API が起動している（Port 8080）
- [ ] Mobile BFF が起動している（Port 8081）
- [ ] テスト用ユーザーアカウント（user001 / password123）でAPIログインが成功する
- [ ] テスト用データ（商品20件以上）が登録されている
- [ ] Windows デスクトップアプリのビルド済み実行ファイルが準備されている
