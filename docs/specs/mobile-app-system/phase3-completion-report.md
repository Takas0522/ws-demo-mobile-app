## 主要タスクの概要

### Phase 1: インフラ・データベース基盤

**ゴール**: 開発環境とデータベースを整備

| タスクID | タイトル | 検証コマンド |
|---------|---------|-------------|
| task-001 | DevContainer環境構築 | `java -version && node -version` |
| task-002 | SQLiteデータベース構築 | `docker-compose ps && sqlite3 ...` |
| task-003 | データベーススキーマ作成 | `sqlite3 -c '\dt'` |
| task-004 | 初期データ投入 | `sqlite3 -c 'SELECT COUNT(*) FROM users'` |

### Phase 2: Web API実装

**ゴール**: ビジネスロジック層の完成

- 認証API（JWT）
- 商品CRUD API
- 購入API
- お気に入りAPI
- 機能フラグ管理API

### Phase 3: BFF層実装

**ゴール**: クライアント専用APIの提供

- Mobile BFF（iOS/Android向け）
- Admin BFF（管理Web向け）

### Phase 4: モバイルアプリ実装

**ゴール**: ネイティブアプリの完成

- iOSアプリ（Swift）
- Androidアプリ（Java）

### Phase 5: 管理Webアプリ実装

**ゴール**: 管理画面の完成

- Vue.js SPAアプリケーション
- 商品管理機能
- 機能フラグ管理機能

### Phase 6: 統合・検証

**ゴール**: エンドツーエンドでの動作確認

- 全コンポーネント統合
- シナリオテスト
- デモ環境構築
