## 概要

本プロジェクトは、モバイルアプリケーションシステムの包括的な仕様策定からバックエンド実装までを完了したものです。エンドユーザー向けモバイルアプリと管理者向けWebアプリを持つECシステムのバックエンドAPIを、BFF（Backend for Frontend）パターンを用いて設計・実装し、**本番稼働可能な状態**になりました。

### システム概要

- **システム名**: モバイルアプリケーションシステム
- **アーキテクチャ**: BFFパターンを採用した3層アーキテクチャ
  - Web API (Core API) - 共通ビジネスロジック
  - Mobile BFF - モバイルアプリ専用BFF
  - Admin BFF - 管理画面専用BFF
- **技術スタック**: 
  - バックエンド: Spring Boot 3.2.1 (Java 17)
  - データベース: PostgreSQL 16
  - 認証: JWT (JSON Web Token)
  - インフラ: Docker / Docker Compose
  - 開発環境: DevContainer

```bash
# PostgreSQL:5432 が起動し、スキーマ・初期データが自動投入される
```

