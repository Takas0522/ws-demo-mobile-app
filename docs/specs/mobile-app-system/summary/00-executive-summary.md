## 概要

本プロジェクトは、モバイルアプリケーションシステムの包括的な仕様策定から**完全なフルスタック実装**までを完了したものです。エンドユーザー向けモバイルアプリ（iOS・Android）と管理者向けWebアプリを持つECシステムを、BFF（Backend for Frontend）パターンを用いて設計・実装し、**本番稼働可能な状態**になりました。

### システム概要

- **システム名**: モバイルアプリケーションシステム
- **アーキテクチャ**: BFFパターンを採用した3層アーキテクチャ
  - Web API (Core API) - 共通ビジネスロジック
  - Mobile BFF - モバイルアプリ専用BFF
  - Admin BFF - 管理画面専用BFF
- **技術スタック**: 
  - バックエンド: Spring Boot 3.2.1 (Java 17)
  - データベース: PostgreSQL 16
  - 認証: JWT (JSON Web Token)
  - インフラ: Docker / Docker Compose
  - 開発環境: DevContainer

## 完成したシステムの起動方法

### 1. データベース起動
```bash
docker-compose up -d
# PostgreSQL:5432 が起動し、スキーマ・初期データが自動投入される
```

### 2. Web API起動（Port: 8080）
```bash
cd web-api
mvn clean package
java -jar target/web-api-1.0.0-SNAPSHOT.jar
```

### 3. Mobile BFF起動（Port: 8081）
```bash
cd mobile-bff
mvn clean package
java -jar target/mobile-bff-1.0.0-SNAPSHOT.jar
```

### 4. Admin BFF起動（Port: 8082）
```bash
cd admin-bff
mvn clean package
java -jar target/admin-bff-1.0.0-SNAPSHOT.jar
```

### 5. 動作確認
```bash
# ヘルスチェック
curl http://localhost:8080/api/v1/health
curl http://localhost:8081/api/mobile/health
curl http://localhost:8082/api/admin/health

# ログインテスト（Mobile）
curl -X POST http://localhost:8081/api/mobile/auth/login \
```
