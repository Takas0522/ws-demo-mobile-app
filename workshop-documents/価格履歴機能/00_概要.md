## 機能概要

商品の価格変更履歴を参照する機能を追加する。
管理者が商品情報を更新し価格が変更された場合に `product_price_history` テーブルに自動記録されたデータを、モバイルアプリ（Android / iOS）から参照できるようにする。

## 改修対象

Web API（API-020）は別部隊にて改修済み。本改修ではモバイル BFF と、モバイルアプリ（Android / iOS）のレイアウト変更を実施する。

| #   | 改修対象       | 改修内容                                             |
| --- | -------------- | ---------------------------------------------------- |
| 1   | Mobile BFF     | 価格履歴取得エンドポイントの追加（Web API への転送） |
| 2   | Android アプリ | 商品詳細画面への価格履歴表示セクション追加           |
| 3   | iOS アプリ     | 商品詳細画面への価格履歴表示セクション追加           |

## API フロー

```
モバイルアプリ
  → GET /api/mobile/products/{id}/price-history
    → Mobile BFF（転送）
      → GET /api/v1/products/{id}/price-history
        → Web API（改修済み）
```
